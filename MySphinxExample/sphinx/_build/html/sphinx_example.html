map_range(number: float, in_min: float, in_max: float, out_min: float, out_max: float, constrained: bool = True) → float¶

Maps a value from one range to another.

This function takes a value within an input range and maps it to the equivalent value within an output range, maintaining the relative position of the value within the range.

Parameters:

  • number (float) – The value to be mapped.

  • in_min (float) – The minimum value of the input range.

  • in_max (float) – The maximum value of the input range.

  • out_min (float) – The minimum value of the output range.

  • out_max (float) – The maximum value of the output range.

  • constrained (bool) – If True, the mapped value is constrained to the output range; default is True.

Returns:

The mapped value.

float
