class TempSensor(name, unit='F')¶

Bases: sensors.Sensor

The temperature sensor class.

Provides access to the connected temperature sensor.

Supported units are “F” (Fahrenheit), “C” (Celsius), and “K” (Kelvin) with “F” being the default unit.
